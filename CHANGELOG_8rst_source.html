12 1.1.0 (2021-01-18)
13 ------------------
14 
15 * Use Eigen Cholesky decomposition instead of a custom one, detect
16  semi-definite Hessians. This change introduces dependency on Eigen 3.3, use
17  older version of `qpmad` if not available.
18 * Refactor API to avoid instantiation of dynamic Eigen matrices and vectors.
19 * Allow specification of scalar type and problem dimensions at compilation time.
20 * The source code is now dependent on C++11 features.
21 * Added methods that expose number of iterations and dual variables.
22 * Added optional hot-starting with inverted Cholesky factor.
23 * DANGER: Solver does not perform Cholesky factorization in some trivial cases
24  anymore, i.e. the Hessian is not necessarily modified. Use `getHessianType()`
25  to get the correct Hessian type when using solver with constant Hessian.
26 * DANGER: Solver does not return error codes anymore, exceptions are used
27  instead in order to make error handling consistent (some errors used to be
28  exceptions in older versions as well). The return codes are: OK,
29  MAXIMAL_NUMBER_OF_ITERATIONS.
